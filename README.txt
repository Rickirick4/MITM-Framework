Usage:
There are 2 options to use this framework;

-t => Target ip address
--target => Target ip address

-g => Target Gateway ip address
--gateway => Target Gateway ip address

Example;
python3 MITM Framework.py -t 0.0.0.0 -g 0.0.0.0
python3 MITM Framework.py -target 0.0.0.0 -gateway 0.0.0.0
